# Superview Frontend

React + TypeScript + Ant Design

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

产物在 `dist/`

## 技术栈

- React 18
- TypeScript
- Ant Design
- Vite
- Zustand (状态管理)
- Axios (HTTP)

## 目录

```
src/
  ├── api/          # API 客户端
  ├── pages/        # 页面组件
  ├── layouts/      # 布局
  ├── hooks/        # 自定义 hooks
  ├── store/        # 状态管理
  └── types/        # TypeScript 类型
```
